Setup Steps

  1. Launch EC2 & Elastic IPUbuntu 20.04 instance, associated Elastic IP 113.50.212.235
  2. Connect via SSHOpened terminal, connected using the Elastic IP address
  3. Update & Upgradesudo apt update and sudo apt upgrade
  4. Install Apachesudo apt install apache2
  5. Install PHP & MySQLPHP runtime, MySQL connector, and MySQL server
  6. Configure MySQLCreated wp database, wp_user, granted all privileges
  7. Install WordPressDownloaded, unzipped into /var/www/html, restarted Apache
  8. Configure Apache Virtual HostSet DocumentRoot, enabled rewrite module
  9. Link Domain via Route 53Hosted zone, A record, updated GoDaddy nameservers
  10. Enable SSL with Certbotsudo certbot --apache -d cybershieldme.info
